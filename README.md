# axiom · teleprompter

Single-page teleprompter. Total black backdrop, Liquid Glass sheet, scrolling script.

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

`bash ship.sh` runs the full pipeline: git init, push to GitHub, deploy to Netlify.

## Shortcuts

- `Space` — play / pause
- `R` — restart
- `↑` / `↓` — speed
- `+` / `−` — font size
- `F` — full-screen
- `M` — mirror
- `Esc` — pause
